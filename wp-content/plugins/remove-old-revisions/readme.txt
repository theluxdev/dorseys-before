This plugin is able to automatically remove revisions older than n days. It is available in english, german, french, spanish.

== Installation ==

1. Copy the folder "remove-old-revisions" to the plugin folder
2. Activate the plugin
3. Head to Settings -> Writing to setup the plugins options

== Screenshots ==
